# 關於Hami書城

## 開始問題之前

Hami書城產品服務，主要分為兩大項：

一、月租收費制

❶月讀包，月租收費制，付費期間可暢讀月讀包架上報紙、雜誌等所有書刊，及使用App內聽書服務，但付費效期終止後，則無法再閱讀從月讀包下載之書刊。（建議有大量閱讀書報雜誌習慣的用戶月租訂閱）

❷藏愛包，月租收費制，付費期間可暢讀藏愛包架上言情小說，包含狗屋、果樹、禾馬、新月等知名品牌 ，以言情小說為主，付費效期終止後，則無法再閱讀從藏愛包下載之小說。

<mark style="color:red;">**※月讀包及藏愛包是兩種不同的月租服務，讀者可依閱讀需求，選擇服務方案。**</mark>

二、電子書購買

購買後於Hami書城平台營運期間可永久使用本平台閱讀所購買之電子書刊內容。（建議給有收藏書刊習慣，且有重複翻閱書刊的用戶採用購買方式）

使用或訂購書城產品前均須先登入(或註冊)Hami書城會員。
