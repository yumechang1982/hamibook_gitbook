# 如何刪除我的個人資料？

Hami書城是由「春水堂科技娛樂股份有限公司 」營運，凡Hami書城用戶，包含Hami書城官網、Hamibook App等註冊會員或訪客。均適用『[Hami書城服務個人資料蒐集告知聲明](https://www.hamibook.com.tw/Homes/action/1/1243)』

當您註冊或第三方登入方式使用「Hami書城」應用程式服務時，均會使用一組Email信箱作為書城帳號，此帳號於Hami書城應用程式中的購買紀錄、閱讀紀錄等均將同步儲存於Hami書城系統，以便於帳號進行跨裝置使用「Hami書城」服務時可同步取得此帳號之閱讀權益。

若您需要將此書城帳號(Email)從Hami書城系統刪除，為確保您是該帳號使用者本人，**請透過該Email信箱寫信**至Hami書城客服信箱（hamibook@kland.com.tw）提出刪除帳號請求。

提出申請後，將由客服人員再次核對申請者為該帳號本人後，**您所使用的Hami書城帳號(Email) 將於十個工作日內於Hami書城系統刪除**。<mark style="color:red;">**提醒您，帳號資料一經刪除，其帳號下之購買紀錄、閱讀紀錄等資料將無法回朔及恢復。申請人不得於事後要求返還相關閱讀權益。**</mark>
