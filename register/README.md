# 書城會員說明

## 登入會員的方式有？

我們提供了多種方式讓您可以快速登入Hami書城會員，但請務必記得您的登入方式，您選擇的登入方式不同，將被視為不同的使用者。

若您已是中華電信會員，建議您可直接採用「中華電信門號」或「中華電信多元帳號」擇一方式登入，好處是您無須再記憶另一組密碼。**＊特別提醒！若您僅是中華電信相關服務客戶，但未註冊成為中華電信會員，仍須先至中華電信會員中心完成會員登入，才可使用上述兩種方式登入Hami書城服務。**[**按此前往中華電信會員中心**](https://member.cht.com.tw/MyMemberCenter/index.jsp)

若您非中華電信會員，則建議您可採用第三方帳號進行登入，目前提供的第三方快速登入方式，包含：**Google 帳號**、**Facebook帳號**、**Apple帳號**，同樣能讓您少記憶一組登入密碼，但須請記得您的選擇的登入方式。若使用Google登入、又選擇用Facebook登入，將被視為兩個不同的帳號。

若您不想採用第三方帳號快速登入，則可選擇先完成註冊會員步驟，填寫Email 及密碼後，並完成Email認證，之後就能採用「電子郵件帳號」方式登入。
