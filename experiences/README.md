# 關於月讀包兌換券

## 什麼是月讀包兌換券？

**月讀包兌換券**，是指一組兌換序號。持有序號可以透過Hami書城網站的兌換專區，或直接透過 Hamibook App內兌換月讀包服務。

[月讀包服務兌換網頁▶](https://www.hamibook.com.tw/Experiences)

兌換券抵用辦法：&#x20;

* 一次兌換限抵用一組序號。
* **兌換券限未申租用戶使用，**&#x82E5;您已擁有月讀包服務，恕無法進行兌換。
* 序號一經使用，則此序號失效，恕不補發，亦不得要求折換現金。
* 兌換券請於期限內使用，逾期即視同自動放棄。

### 官網兌換指引

❶用戶註冊成為Hami書城會員，登入會員帳號，

點選右上方「兌換券 」進入序號兌換頁面。

<img src="../.gitbook/assets/兌換券Step1 (1).png" alt="" data-size="original">&#x20;

❷在「序號兌換」欄位輸入序號，點選兌換鈕。序號大小寫需正確填寫，方能開通服務。

<img src="../.gitbook/assets/兌換券Step2.png" alt="" data-size="original">&#x20;

❸  顯示「兌換成功」，表示您已完成兌換。

### App 內兌換指引

❶開啟HamiBookAPP，右上方齒輪「設定」專區中選擇 「前往兌換月租體驗序號」。



❷在「兌換序號」欄位輸入序號，勾選\[我已閱讀並同意服務使用條款]，點選\[兌換]按鈕。序號大小寫需正確填寫，方能開通服務。



❸  顯示「兌換成功」，表示您已完成兌換。

## 如何獲取兌換券？

您可以透過合作通路活動，或是書城會員獎勵活動，獲得兌換券，亦可透過合作通路：7-ELEVEN 、全家便利商店、超商通路直接購買序號。
